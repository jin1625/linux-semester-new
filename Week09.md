# Week 9 - MLflow로 머신러닝 실험 기록 및 비교

## 1. 이번 주 목표
- MLflow 설치 및 환경 구성
- 모델 학습 결과 자동 기록
- 모델 간 성능 비교

## 2. 설치 / 환경 구성
- Python 버전 이슈 → week9-mlflow 환경 생성
- pip 업그레이드 후 MLflow 설치
- 동일 환경에서 JupyterLab 설치

## 3. 실습 과정
- 당뇨병 예측 선형회귀 모델 학습
- 기본 모델 + 규제 모델(Ridge 등) 비교
- MLflow UI에서 run 기록 확인
- 파라미터, MSE, 그래프 비교

## 4. 문제 & 해결
- Python 버전 충돌 → 새 환경 생성으로 해결
- MLflow + JupyterLab 조합이 매우 안정적

## 5. 리눅스 사용 소감
- 환경 분리가 아주 쉬움
- MLflow 서버 실행 안정적
- 실험 관리가 깔끔하게 구조화됨
