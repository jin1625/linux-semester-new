# Week 3 - VS Code + Conda + Git 연동

## 1. 이번 주 목표
- VS Code 설치
- Conda 환경과 VS Code 연동S
- Git과 GitHub 연결

## 2. 설치/환경 구성
- 터미널 명령어로 VS Code 설치
- VS Code 터미널에서 conda 환경 활성화
- pandas, matplotlib 등 기본 라이브러리 설치
- settings.json 수정하여 VS Code가 항상 해당 환경을 사용하도록 설정

## 3. 실습 과정
- pandas로 간단한 DataFrame 만든 뒤 시각화
- GitHub 리포 생성 후 VS Code와 연동
- 커밋 & 푸시 테스트 완료

## 4. 문제 & 해결
- 특이사항 없음 (리눅스에서 Git 기본 설치되어 매우 편리)

## 5. 리눅스 사용 소감
- VS Code 반응이 빠르고 딜레이가 거의 없음
- conda 환경 인식이 윈도우보다 안정적
- Git 설치 없이 바로 사용 가능해서 편함
